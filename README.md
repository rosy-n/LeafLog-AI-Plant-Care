# LeafLog

LeafLog is an AI-powered plant care app for registering plants, managing care schedules, writing growth diaries, generating AI plant characters, and supporting plant diagnosis.

## MVP Features

- Login and signup
- Home screen
- Plant registration
- My plants list
- Plant detail screen
- Care schedule
- Growth diary
- Photo upload and gallery
- AI plant character generation prototype
- Plant diagnosis prototype

## Planned Features

- Plant information tab
- Persona chat
- Sensor dashboard
- Gamification
- Memorial space
- Community feature

## Tech Stack

### Mobile

- React Native
- Expo
- TypeScript

### Backend

- FastAPI
- PostgreSQL

### AI

- Plant species identification
- AI character generation
- Plant diagnosis
- Persona chat
